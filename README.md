# MACD Multi-Frame

MACD Multi-Timeframe is an MTF indicator for MT4 and MT5 developed by EarnForex.com. It is based on the standard MACD (Moving Average Convergence Divergence) indicator. It reads the data from the standard MACD on multiple timferames and displays a single panel on the current chart.

The indicator can work with multiple MACD signals and their combinations: rising/falling, above/below zero, above/below signal line. It supports all kinds of alerts.
